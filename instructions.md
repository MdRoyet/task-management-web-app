# Project Guidelines

## Tech Stack
- **Backend:** Node.js, Express, Socket.io, Prisma, PostgreSQL
- **Frontend:** React (Vite)

## Architecture & Security Rules
- **Never** hardcode secrets. Always use environment variables or a secrets manager.
- Always adhere to a strict **MVC (Model-View-Controller)** architecture.
- **Request user review** before running any destructive database commands.
